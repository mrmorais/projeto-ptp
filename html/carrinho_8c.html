void carrinho_addProduct ( Carrinho *  cart,
Produto *  p 
)

Adiciona produto ao Carrinho.

Esta função adiciona no Carrinho cart a referência para o Produto p.
Este produto é inserido na posição next_index (que é posteriormente incrementada em uma unidade) do products[] na Struct Carrinho.

Parameters
*cartPonteiro para o carrinho de produtos, no qual o produto deve ser inserido.
*pPonteiro para o produto que deve ser anexado ao final do array de produtos do *cart

Esta rotina void imprime, na saída padrão, as informações de cada produto adicionado ao carrinho. E, por fim, imprime o preço total a ser pago, calculado por carrinho_totalPrice(Carrinho *cart).
