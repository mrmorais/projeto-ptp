Busca produtos por tipo de pesquisa.

Faz uma leitura de produtos persistidos e seleciona os produtos com as especificações dadas. Os tipos de busca, definidos em int search_type, são

TipoFiltro emValor Descrição
ID p.id 1 Buscar produto por ID
Nome p.descricao 2 Buscar produto por Nome
Fornecedor p.fornecedor 3 Produtos de um fornecedor

O termo de busca vem determinado no parâmetro *p

search_typeCritério de busca
*pReferência ao Produto com o termo de busca determinado
